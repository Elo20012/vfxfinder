VFXFINDER

The goal of this website is to help VFX Artists to find inspirations quickly.
I love creating VFX Videos myself so I wanted to stop using all those YouTube playlists.
Using simple HTML, CSS, JavaScript and JSON Data and with the help of ChatGPT I was able to create this website.
Inspired by the websitesakugabooru.com, I decided to create a space for 3D where any Artist or not, can participate andadd a videowith a link to your portfolio if you would like (your name with hyperlink will be listed below).
To keep this website free, I will not use any Ads nor Servers. I use a simple JSON file to add videos, so please be patient if your video doesn't appear yet.
